- Tool: Cypress
- IDE: VSCODE
- Language: JAVAScript

## Prerequisite:
1. Need to install jdk 1.8
2. Need good internet connectivity
3. Node JS
4. VSCODE

## Run the automation script:
1. Open VS CODE project
2. Type this command: npx cypress open
